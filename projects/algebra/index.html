Help

 Algebra is all about symmetry, but with the usual representation of groups (with symbols and numbers)
it is not easy to see and appreciate the beauty and the symmetry of groups.

So, this application helps you see this beauty by generating the Cayley table of the group,
but instead of using symbols it uses colors.

How to use it :

 Order : Enter the order of the group (the number of elements).
Elements : Enter the names you use for the elements (they should be separated by a space - see the example). e.g. : e i j k
Colors : Enter the colors that you want to use (in the order of the corresponding elements) separated by a space. e.g. : Red Green Yellow Blue
Table : Input the cayley table of the group, using the spacing conventions of the example and the elements you defined above.
Print symbols : Use this option to control whether the symbols should be print or not.

For any comments or suggestions, please contact me at : nikolaskaralis@gmail.com
