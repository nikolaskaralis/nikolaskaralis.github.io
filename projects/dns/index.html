DNS Conflicts Project


Introduction

During a conversation with my good friend, Nikos Charitonides (aka Jheremias) he told me his idea,about creating a program that would find all the DNS conflicts (if they exist) under a given subnet.
At first, we were not sure if there would exist any. So, I put together a Python that does exactly that.

How does the program works?

It takes a subnet, let's say '147.102.' and it scans all the IP addresses that it includes.
This means that it scans all IPs between 147.102.0.0 - 147.102.255.255.
It then does a reverse nslookup at all these IPs and gets the domain where they point.
After that it does another nslookup at that domain and then checks for any DNS conflicts.
