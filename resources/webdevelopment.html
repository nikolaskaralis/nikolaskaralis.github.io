  • If you have a form and a submit button and the same php page receives the POST request, then the page doesn't refresh automatically after the submit button is pressed. To force the refresh, place this code somewhere in the if block :

    if ($_SERVER["REQUEST_METHOD"] == "POST") The code to add : $uri = $_SERVER['REQUEST_URI']; echo "<meta http-equiv=refresh content='0'; URL='$uri']'>"; // This is a trick to reload the page after the submit of the form.
