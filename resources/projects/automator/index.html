MacOSX utilities


All the utilities are created with the Automator. You can also use it to edit the applications to fit your needs.
They all work by droping the files you want to convert onto the application icon.

  • CombinePDFs

    • Drop PDF files on the icon and it combines them in one file.   [Download]

  • jpg2pdf

    • Convert image files to PDF files.   [Download]

  • pdf2jpg

    • Convert PDF files to jpg files.   [Download]

  • resizeimages

    • Resize image files to one-third of their size.   [Download]
